# YT Research Tool

A YouTube outlier research and analysis tool.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Stack
- React 18 + Vite
- Tailwind CSS + shadcn/ui
- Zustand
- React Router v6
- Recharts
- YouTube Data API v3
- Multi-provider AI API (Claude, GPT, Gemini, OpenRouter, Groq)
